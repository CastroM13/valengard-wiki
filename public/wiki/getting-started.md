# Getting Started

Welcome to the wiki! This guide will help you get started.

## Creating Articles

To create a new article:

1. Create a new `.md` file in the `public/wiki/` directory
2. Write your content in markdown format
3. Link to it from other articles using `[Link Text](article-name)`

## Linking Between Articles

You can link to other articles using markdown link syntax:

```markdown
[Link Text](article-name)
```

Or with the `.md` extension:

```markdown
[Link Text](article-name.md)
```

Both formats work and will be automatically converted to the correct route.

## Article Naming

- Use lowercase letters and hyphens for article names
- Examples: `getting-started`, `api-reference`, `user-guide`
- Access articles via `/wiki/article-name`
